Оптимізуйте автентифікацію, управління користувачами та забезпечення за допомогою вашого існуючого провайдера ідентичності (IdP).

Google Workspace

Підключіть свій обліковий запис Google Workspace для єдиного входу (SSO) та синхронізації користувачів і груп через SCIM-протокол.

Microsoft Azure Active Directory

Реалізуйте концепцію Zero Trust у вашому каталозі користувачів за допомогою Twingate та Microsoft Azure Active Directory.

OneLogin

Використовуйте OneLogin для автентифікації та синхронізації користувачів і груп через SCIM.

Duo Security

Використовуйте Duo Security як програму MFA для ресурсів, захищених Twingate.

Інфраструктура та автоматизація

Доведіть безпеку Zero Trust до швидкості DevOps за допомогою інтеграції постачальників IAC, конвеєрів CI/CD і облікових записів служб.

Terraform

Kubernetes

Github
